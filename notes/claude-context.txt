## Proje Özeti

**ARCB Wider Updater** - Linux sistemleri için tek komutla çoklu paket yöneticisi güncelleme aracı.

- **Slogan:** "One command. One updater. Zero nonsense."
- **Geliştirici:** ahm3t0t (ahmet@tanrikulu.net)

## Kullanılabilir Komutlar

Kurulumdan sonra 3 komut kullanılabilir:

  guncel      # Türkçe kullanıcılar için
  updater     # İngilizce kullanıcılar için
  bigfive     # Uluslararası/marka ismi

Hepsi aynı script'i çalıştırır (symlink).

---

## Desteklenen Paket Yöneticileri (BigFive)

| Paket Yöneticisi | Dağıtımlar | Fonksiyon |
|------------------|------------|-----------|
| APT | Debian, Ubuntu, Zorin, Mint | update_apt() |
| DNF | Fedora, RHEL, CentOS | update_dnf() |
| Pacman | Arch, Manjaro, EndeavourOS | update_pacman() |
| Zypper | openSUSE Leap/Tumbleweed | update_zypper() |
| APK | Alpine Linux | update_apk() |

+ Flatpak, Snap, Firmware (fwupdmgr)

  # Fedora test
  docker run --rm -v "$(pwd):/app" fedora:40 bash -c "dnf install -y curl && bash /app/install.sh && guncel --help"

  # Arch test
  docker run --rm -v "$(pwd):/app" archlinux:latest bash -c "pacman -Sy --noconfirm curl && bash /app/install.sh && guncel --help"

  # openSUSE test
  docker run --rm -v "$(pwd):/app" opensuse/tumbleweed:latest bash -c "zypper install -y curl gawk && bash /app/install.sh && guncel --help"

  # Alpine test
  docker run --rm -v "$(pwd):/app" alpine:3.20 sh -c "apk add bash curl && bash /app/install.sh && guncel --help"
