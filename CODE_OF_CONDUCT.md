# Code of Conduct

## Our Pledge
We want **ARCB Wider Updater** to be a welcoming, friendly, and harassment-free space for everyone.

## Be Nice, Be Professional
- Be respectful and constructive.
- Assume good intent, but keep discussions factual.
- Critique code, not people.
- Keep things safe: do not post secrets, tokens, private keys, or sensitive logs.

## Unacceptable Behavior
- Harassment, hate speech, threats, or discrimination
- Personal attacks, trolling, or deliberate disruption
- Publishing private information (doxxing) or credentials
- Sexualized language or imagery, unwanted advances

## Scope
This policy applies in:
- GitHub Issues, Pull Requests, Discussions
- Project communication related to this repository

## Enforcement
Maintainers may take action to keep the community safe and productive:
- Warning
- Content removal
- Temporary or permanent ban

## Reporting
If you experience or witness unacceptable behavior:
- Preferably open a GitHub issue titled **"CoC Report"** and mark it as **confidential** in the description (avoid sensitive data).
- Or contact the maintainer via email: **ahmet@tanrikulu.net**

We appreciate your help in making this project better for everyone.
